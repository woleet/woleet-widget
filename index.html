Data Anchoring on Bitcoin proof verifier

Use this tool to verify the timestamp and integrity of files anchored on the Bitcoin blockchain using any Chainpoint 1.x compatible tool.

Note: if the file was anchored publicly using the Woleet platform, then you don't need to provide the anchoring receipt (as it is retieved live from the Woleet platform). Otherwise, you'll be asked to provide the Chainpoint 1.x anchoring receipt.
